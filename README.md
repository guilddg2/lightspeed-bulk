# lightspeed-bulk

Numerical and analytical tools supporting the paper:

**Quantum Mechanics and Gravity from Light-Speed Bulk Intersections: A Geometric Unification**  
David Guild, July 2025

## Contents

- `bulk_solver.py`: 5D bulk evolution using Discontinuous Galerkin and spectral methods
- `holographic_qm.py`: Derivation and simulation of holographic Schr√∂dinger equation
- `cosmology_analysis.py`: CMB power spectrum and inflation parameter analysis
- `echo_search.py`: Gravitational wave echo template generator and SNR estimator
- `galaxy_dynamics.py`: Pipeline for testing dark matter prediction with Gaia DR4

## Dependencies

Install with:

```
pip install -r requirements.txt
```
